Welcome to Coding Game!
Instructions:
 You are a lone robot on this planet. Your quest is to find the supply drop sent from your home planet by navigating the troublesome terrain.
 Use the following commands and structures to complete the tasks:
  Commands:
   -up(amount)
   -down(amount)
   -right(amount)
   -left(amount)

   -function <code>
  Defining a function:
   After "function ", write your lines of code separated by a ';' inside curly brackets
   To write a function 'test' that will move up 3 and right 4, write:
    function test {
     up(3);
     right(4);
    }
